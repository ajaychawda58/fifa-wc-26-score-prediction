### Project

This project aims to create a score prediction engine for FIFA world cup 2026 matches. The engine should be able to predict the score of a match based on historical data of the teams playing in the match.    


### Goals

It should be able to predict the score of a match based on historical data of the teams playing in the match. It should also be able to predict the probability of each team winning the match and the probability of each team scoring a certain number of goals.  


### Steps

1. Identify sources of FIFA world cup 2026 match data
2. Collect historical data of the teams playing in the match and store in mark down file
3. Collect Playerwise data of each teams selected squad and store player profile in a markdown file
4. Collect previous 20 matches data of each team and their score in a markdown file.
5. Preprocess the data to make it suitable for the model
6. Train the model using the preprocessed data
7. Evaluate the model's performance
8. Deploy the model to predict the score of a match
9. Continuously monitor and improve the model's performance
10. Take into account the matches that have already happend in the world cup and tune the model.
11. Display the predicted score and actual score in a github page.
